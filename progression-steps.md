## Progression Steps
```plain text
    Generate QA --> Read -> Learn and Memorize -> Mark Unknown Vocabularies
```
### Step Details
- Genearate QA
    - Prepare the initial QA(s) of all the sections of the chapter.
    - Use `Gemini 3 Pro`, `DeepSeek v3.2` or `Claude Opus 4.5-Think` from `outlier.ai`.
    - Check the generated QA pairs and perform changes required.
- Read -> Learn and Memorize includes
	- Completing one section
	- Understand and implement the commands in the practice lab
	- Check whether the mentioned thing in the book are correct or not.
	- If not, find out what went wrong and the correct thing.
	- Note down those.
- Mark Unknown Vocabularies:
    - Highlight the vocabs to learn on a later time period.
