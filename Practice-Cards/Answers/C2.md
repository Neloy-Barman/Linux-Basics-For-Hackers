hed - Install `snort` from following directions from `https://www.kali.org/tools/snort/`
  ```bash
    sudo apt install snort
  ```
  - If it raises issues, follow the directives from CLI.
    ```bash
      sudo apt update
    ```
- **Note:** _We will use `/etc/snort/snort.lua` as the file to execute commands._
- View file from the beginning
  ```bash
    head /etc/snort/snort.lua
  ```
- View first 20 lines of the file
  ```bash
    head -20 /etc/snort/snort.lua
  ```
- View file from the ending
  ```bash
    tail /etc/snort/snort.lua
  ```
- View last 20 lines of the file
  ```bash
    tail -20 /etc/snort/snort.lua
  ```
- Number Lines of the file
  ```bash
    nl /etc/snort/snort.lua
  ```
- Find all the lines with word `output`
  ```bash
    cat /etc/snort/snort.lua | grep output
  ```
- View man page of any of the command
  ```bash
    man tail
  ```
- Hacker Challenge: Display the 5 lines immediately before the line: `-- payload_injector = { }`
  - **Note:** _Don't count the empty spaces as lines._
  - 1st Approach
    ```bash
      nl /etc/snort/snort.lua | grep payload
      tail -n+7 /etc/snort/snort.lua | head -n 5
    ```
  - 2nd Approach
- Find an Replace 
  - Replace the word `tcp` with `TCP`
    - Globally
      ```bash
        sed s/tcp/TCP/g /etc/snort/snort.lua > snort2.lua
      ```
    - 1st Occurence
      ```bash
        sed s/tcp/TCP/ /etc/snort/snort.lua > snort2.lua
      ```
    - 2nd Occurence
      ```bash
        sed s/tcp/TCP/2 /etc/snort/snort.lua > snort2.lua
      ```
  - Save the file `snort2.lua` 
  - Remove `snort2.lua`
    ```bash
      rm snort2.lua
    ```
- View file using Page System
  - Normal
    ```bash
      more /etc/snort/snort.lua
    ```
  - Along with Searching Occurences of particular word
    ```bash
      less /etc/snort/snort.lua
    ```
    - Press key and type `output` for search
      ```bash
        /output
      ```
    - Press Key to go to next occurences 
      ```bash
        n
      ```
