1. Finding Yourself in the Current Directory
   ```bash
   pwd
   ```
2. Checking Logged-in User
   ```bash
   whoami
   ```
3. Changing Directories Forward - `etc`
   ```bash
   cd /etc
   ```
4. Changing Directories Backward
   1. `One` step
      ```bash
      cd ..
      ```
   2. `Multiple` Steps
      1. Go to `/etc/fonts` and come back
         ```bash
         cd ../..
         ```
5. Listing the Contents of a Directory
   1. Normal
      ```bash
      ls
      ```
   2. In-Depth Details
      ```bash
      ls -l
      ```
   3. Hidden Ones
      ```bash
      ls -la
      ```
6. Getting Help Regarding Applications - `aircrack-ng`, `nmap`
   ```bash
   aircrack-ng --help
   nmap -h
   nmap -?
   ```
7. Referencing Manual Pages - `aircrack-ng`
   ```bash
   man aircrack-ng
   ```
8. Finding Stuff
   1. Search for `aircrack-ng`
      ```bash
      locate aircrack-ng
      ```
   2. Find Binaries of `aircrack-ng`
      ```bash
      whereis aircrack-ng
      ```
   3. More Powerful Searches
      1. Search for file `apache2` in root (/).
         ```bash
         find / -type f -name apache2
         ```
      2. Search for file `apache2` in /etc.
         ```bash
         find /etc -type f -name apache2
         ```
      3. Generic Search for `apache2.conf` in /etc.
         ```bash
         find /etc -type f -name apache2.*
         ```
      4. Generic Search for cat, hat, bat, and what
         ```bash
         find /etc -type f -name *at
         ```
   4. Filtering
      1. Search for the running services.
         ```bash
         ps aux
         ```
      2. Find out if `apache2` is running.
         ```bash
         ps aux | grep apache2
         ```
9. Create a Directory
   1. Name: `newdirectory`
      ```bash
      mkdir newdirectory
      ```
   2. Perform next steps within the new directory `newdirectory`.
10. Create a File
    1. 1st Method
       1. File Name: `hackingskills`
          ```bash
          cat > hackingskills
          Hacking is the most valuable skill set of the 21st century!
          ```
       2. View File Content
          ```bash
          cat hackingskills
          ```
       3. Append Content
          ```bash
          cat >> hackingskills
          Everyone should learn Hacking.
          ```
       4. Overwrite File
          ```bash
          cat > hackingskills
          Everyone working in IT without the knowledge of hacking is in Darkness.
          ```
    2. 2nd Method
       1. File Name: `newfile`
          ```bash
          touch newfile
          ```
11. Copying a File
    1. Come back one step backward from `newdirectory`.
       ```bash
       cd ..
       ```
    2. Create file `oldfile`.
       ```bash
       touch oldfile
       ```
    3. Copy to `newdirectory` with name `newfile`
       ```bash
       cp oldfile newdirectory/newfile
       ```
12. Renaming a File
    1. Old Name: `newfile`
    2. New Name: `newfile2`
       ```bash
       mv newfile newfile2
       ```
13. Removing a File
    1. Name: `newfile2`
       ```bash
       rm newfile2
       ```
14. Removing a Directory
    1. Name: `newdirectory`
    2. Empty Directory
       ```bash
       rmdir newdirectory
       ```
    3. Non-Empty Directory
       ```bash
       rm -r newdirectory
       ```
