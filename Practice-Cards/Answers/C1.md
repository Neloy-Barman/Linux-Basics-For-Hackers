- Finding Yourself in the current Directory
  ```bash
    pwd
  ```
- Checking Logged in User
  ```bash
    whomai
  ```
- Changing Directories Forward - `etc`
  ```bash
    cd /etc
  ``` 
- Changing Directories Backward 
  - `One` step
    ```bash
      cd ..
    ``` 
  - `Multiple` Steps
    - Go to `/etc/fonts` and come back
    ```bash
      cd ../..
    ``` 
- Listing the Contents of a Directory
  - Normal
    ```bash
      ls
    ``` 
  - In-Depth Details
    ```bash
      ls -l
    ``` 
  - Hidden Ones
    ```bash
      ls -la
    ``` 
- Getting Help regarding Application - `aircrack-ng`, `nmap`
    ```bash
    aircrack-ng --help
    namp -h
    nmap -?
  ``` 
- Referencing Manual Pages - `aircrack-ng`
  ```bash
    man aircrack-ng
  ``` 
- Finding Stuff
  - Search for `aircrack-ng`
    ```bash
      locate aircrack-ng
    ``` 
  - Find Binaries of `aircrack-ng`
    ```bash
      whereis aircrack-ng
    ``` 
  - More Powerful Searches
    - Search for file `apache2` in root(/).
      ```bash
        find / -type f -name apache2
      ``` 
    - Search for file `apache2` in /etc.
      ```bash
        find /etc -type f -name apache2
      ``` 
    - Generic Search for `apache2.conf` in /etc.
      ```bash
        find /etc -type f -name apache2.*
      ``` 
    - Generic Search for cat,hat, bat and what
      ```bash
        find /etc -type f -name *at
      ``` 
  - Filtering
    - Search for the running services.
      ```bash
        ps aux
      ``` 
    - Find out if `apache2` is runnning.
      ```bash
        ps aux | grep apache2
      ``` 
-  Create a Directory
   -  Name: `newdirectory`
      ```bash
        mkdir newdirectory
      ``` 
  - Perform next steps within the new dir `newdirectory`.
- Create a File
  - 1st Method
    - File Name: `hackingskills`
      ```bash
        cat > hackingskills
        Hacking is the most valuable skill set of the 21st century!
      ``` 
    - View File Content
      ```bash
        cat hackingskills
      ``` 
    - Append Content
      ```bash
        cat >> hackingskills
        Everyone should learn Hacking.
      ``` 
    - Overwrite File
      ```bash
        cat > hackingskills
        Everyone working in IT without the knowledge of hacking is in Darkness.
      ``` 
  - 2nd Method
    - File Name: `newfile`
      ```bash
        touch newfile
      ``` 
- Copying a file
  - Come back one step backward from `newdirectory`.
    ```bash
      cd ..
    ```
  - Create file `oldfile`.
    ```bash
      touch oldfile
    ```
  - Copy to `/newdirectory` with name `newfile`
    ```bash
      cp oldfile /newdirectory/newfile
    ``` 
- Renaming a File
  - Old Name `newfile`
  - New Name `newfile2`
    ```bash
      mv newfile newfile2
    ``` 
- Removing a File
  - Name: `newfile2`
    ```bash
      rm newfile2
    ``` 
- Removing a Directory
  - Name: `newdirectory`
  - Empty Directory
    ```bash
      rmdir newdirectory
    ``` 
  - Non-Empty Directory
    ```bash
      rm -r newdirectory
    ```
